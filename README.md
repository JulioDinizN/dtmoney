# dtmoney

dtmoney - Next.js webapp made in a coding event with the intent of learning more and practice React development. It is an application where the main goal is to be a money managment tool and
help the user visualize better where they earnings are being spent.

The goal of the webapp was also to learn more about build tools such as webpack and how a react project is bootstraped
